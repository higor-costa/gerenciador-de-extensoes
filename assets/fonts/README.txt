Fonte Variável Noto Sans
=======================

Este download contém Noto Sans como fontes variáveis e estáticas.

Noto Sans é uma fonte variável com os seguintes eixos:
largura
largura

Isso significa que todos os estilos estão contidos nestes arquivos:
NotoSans-VariableFont_wdth,wght.ttf
NotoSans-Italic-VariableFont_wdth,wght.ttf

Se o seu aplicativo for totalmente compatível com fontes variáveis, agora você pode escolher estilos intermediários
que não estão disponíveis como fontes estáticas. Nem todos os aplicativos suportam fontes variáveis e,
nesses casos, você pode usar os arquivos de fonte estática para Noto Sans:

static/NotoSans-Regular.ttf
static/NotoSans-Medium.ttf
static/NotoSans-Bold.ttf

Começar
-----------

1. Instale os arquivos de fonte que deseja usar

2. Use o seletor de fontes do seu aplicativo para visualizar a família de fontes e todos os
estilos disponíveis
